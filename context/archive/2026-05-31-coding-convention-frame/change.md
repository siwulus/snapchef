# Extensible Coding-Style Conventions for Agentic Coding

Establish an extensible, agent-readable conventions registry under
`docs/reference/conventions/` (one file per style domain + an index), wire it
into `CLAUDE.md` so agents reliably load and obey it, seed it with generic
style rules (arrow functions, functional programming) and Zod naming
conventions, then reconcile existing reference docs and source code into
compliance.

See `plan-brief.md` for the two-pager and `plan.md` for the full plan.
